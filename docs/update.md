    - 토큰 상점에 `끝말잇기 전적 초기화` 아이템이 추가되었습니다.
    - 상자 쿠폰 등록 시 저장소 용량을 무시하고 등록되도록 변경되었습니다.

## 2021.09.24
- 투표 봇
  - 투표 생성, 로그 설정 데이터의 저장 기간이 변경되었습니다.
    - 일반 이용자: 30일 / 프리미엄 이용자: 영구
    - 이로 인해 모든 데이터가 초기화되었습니다.
- 츠리 봇
  - 베팅, 강화 파괴 시 복구할 수 있는 기능이 추가되었습니다.
  - `-츠로필` 명령어에 나왔던 개인 순위 정보가 `-순위 코인` 명령어로 이동되었습니다.
  - `-내설정 배너` 배너 설정 조건이 변경되었습니다.
  - 프로필 공개 설정이 추가되었습니다.
    - `-내설정 상메` 명령어로 상태 메시지에 `-비공개` 문구를 넣으면 계정이 비공개 상태로 전환됩니다.
    - 비공개 상태일 때는 다른 사람이 본인 정보 열람할 수 없고, 송금을 받을 수 없게 됩니다.
  - `-거래` 일부 제한이 변경되었습니다.
    - 최소 거래 제한이 삭제되었습니다.
    - 보유 중인 코인의 70%까지만 거래에 참여할 수 있게 변경되었습니다.
  - `-상점` 칭호와 이용권 구매 시 상점 쿨타임이 돌지 않도록 변경되었습니다.
  - `-상자` `고급`, `스페셜` 상자의 코인 획득량이 대폭 하향 조정되었습니다.
  - 신규 재화 `토큰`이 추가되었습니다.
  - 시즌 초기화 이후 신규 유저에게 적용되었던 오류가 수정되었습니다.
  - `-끝말잇기` 여러 가지 문제가 개선되었습니다.
    - 츠리도 두 글자 이상만 사용 가능하게 수정되었습니다.
    - 한 게임에 단어 25개 이상 사용해야 순위에 기록되도록 변경되었습니다. (연습, 랭크)
    - `-끝말 순위` `단어` 항목이 `사용한 단어`로 변경되었습니다.
    - 끝말잇기 데이터가 초기화되었습니다.
  - `-라이어게임` 주제 선택 오류가 수정되었습니다.

## 2021.08.31
- 투표 봇
  - `=삭제` 99개의 메시지를 삭제할 수 없었던 버그가 수정되었습니다.
  - `=백업` 백업 가능 개수가 1,000건에서 3,000건으로 증가되었습니다.
  - `=로그` 전체적으로 로그 성능이 향상되었습니다.
  - `=음악` 음악 명령어가 삭제되었습니다.
- 츠리 봇
  - `-돈받기` 쉽게 돈을 받을 수 있도록 수정되었습니다.
  - `-인벤토리` 엠블럼 시스템이 추가되었습니다.
  - `-저금` 이제부터 저장소 용량이 최대 저금 코인에 영향을 줍니다.
    - 최대 저금 코인: 저장소 용량의 1만 배
  - `-송금` 송금 최대 코인이 하향 조정되었습니다.
    - 70만 원 → 20만 원, 2,000만 원 → 70만 원
  - `-순위` 각 순위의 우선순위가 변경되었습니다.
    - 변경 전: 재화 → 디스코드 가입일, 변경 후: 재화 → 츠리 봇 가입일
  - `-상자` 일반 상자가 버프 되었습니다.
  - `-방생성` 명령어가 추가되었습니다.
  - `-설정` 명령어가 삭제되었습니다.
  - `-인증` 인증 시 지급되는 배지가 삭제되었고, 엠블럼이 지급되도록 변경되었습니다.
  - 저장소 최대 레벨이 변경되었습니다.
    - 변경 전: Lv.500, 변경 후: Lv.200 / 추후 업데이트를 통해 증가될 예정입니다.
  - 작업실 Premium의 일부 후원자 전용 특전이 종료되었습니다.
    - 종료된 특전: 닉변 무료, 최대 저금 코인 증가
  - 구독형 Premium의 후원자 전용 특전이 종료되었습니다.
  - `-끝말잇기` 시작 메시지에 츠리 봇 닉네임이 아닌, 디스코드 닉네임으로 표기되는 문제가 수정되었습니다.
