### 후원 방식별 후원 금액
- 후원 페이지: https://toss.me/donut
- 츠리 서포터: 10,000원 이상부터 자율
- 토큰(씨앗/잎새/나무): 1,000원/3,000원/5,000원

### 츠리 서포터 후원 특전

1. `츠리 서포터` 칭호 획득
2. 후원자 전용 엠블럼 부여
3. 더 많은 코인 송금 가능
4. 프로필 배너 설정 가능
5. 더 긴 닉네임 설정 가능
6. 더 긴 상태 메시지 설정 가능
7. 봇 관리 서버에서 후원자 `전용 역할` 지급
8. 봇 관리 서버에서 후원자 `전용 채널` 오픈
9. 개발자의 특별 관리 (개발자와 직접 대화 가능)
