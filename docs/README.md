# 츠리 봇

## 안내
- 츠리는 MZ세대를 대상으로 가상 코인을 이용한 아케이드 게임을 지원하는 Discord 챗봇입니다.
- 봇 초대 및 이용 시 츠리의 [이용약관](https://bot.dowon.monster/#/terms)과 [개인정보처리 방침](https://bot.dowon.monster/#/privacy)에 동의하는 것으로 간주됩니다.
- 명령어 가이드는 옆 ( ≡ ) 버튼을 클릭해서 사이드 바를 열어 주세요.

## 공식 디스코드
